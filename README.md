# Numpy_NN

### 通过 Numpy 实现的神经网络

#### 激活环境

##### Windows

```shell
cd ./neural_network/Scripts
```

```shell
./activate
```

##### Linux

```shell
source ./neural_network/Scripts/activate
```

#### 运行程序

##### Windows

```shell
python train.py
```

##### Linux

```shell
./neural_network/Scripts/python.exe train.py
```
