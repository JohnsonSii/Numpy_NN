# Numpy_NN

### 通过 Numpy 实现的神经网络

#### 激活环境

##### Windows

```shell
./neural_network/Scripts/activate
```

##### Linux

```shell
source ./neural_network/Scripts/activate
```

#### 运行程序

```shell
./neural_network/Scripts/python.exe train.py
```

#### 运行交互程序

```shell
pip install PyQt5
```
